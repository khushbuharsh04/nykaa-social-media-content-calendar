# 📅 Nykaa Social Media Content Calendar

## 📌 Overview

This project showcases a **4-week Social Media Content Calendar** created for **Nykaa** as part of my Digital Marketing portfolio. It demonstrates strategic content planning across multiple social media platforms with a balanced mix of promotional, educational, engagement, and brand-building content.

## 🎯 Objective

To develop a structured monthly content strategy that:

- Increases brand awareness
- Drives audience engagement
- Promotes products and campaigns
- Builds customer trust
- Maintains consistent brand communication
